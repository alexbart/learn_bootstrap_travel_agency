# Explore World Travel Agency

A responsive landing page for a fictional travel agency using **Bootstrap 4.6**.

## Features
- Responsive navbar with collapse
- Hero section with overlay
- Top destinations cards with hover effect
- Tours section using Bootstrap 4 collapse/accordion
- Contact form with rounded inputs and hover effects
- Footer with social links
- Fully responsive for desktop, tablet, and mobile

## Technologies Used
- HTML5
- CSS3
- Bootstrap 4.6
- jQuery & Popper.js
